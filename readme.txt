Program byl vytvořen na míru pro zpracování reportů z hovorů na MSTeams pro Team KVK.

Pro správnou funkčnost je potřeba:
1. Ve složce, kde je program umístěn, musí existovat složka s názvem "reports".
2. Do této složky se pak umístí všechny reporty, které chceme zpracovat a musí mít
název "1 (x).csv", tedy v praxi "1 (1).csv", "1 (2).csv", "1 (3).csv", atd.

Po spuštění programu se zpracují všechny správně pojmenované soubory, přejmenují se
a vytvoří se výstupní soubor "summary_report.csv".
